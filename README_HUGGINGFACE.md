# ⬡ THE SWARM ⬡

Multi-Persona Pattern Processor - 8 AI personas analyze your content through different cognitive lenses.

## What Is This?

THE SWARM processes text through 8 specialized AI personas:

### Front Line (Parallel)
- 🔴 **MOTIF** - Theme Hunter
- 🟢 **CATALOG** - Entity Mapper
- 🟡 **PRISM** - Perspective Shifter
- 🔵 **LATTICE** - Pattern Spotter

### Hidden Layer (Sequential Synthesis)
- 🟣 **WEAVER** - Cross-Persona Synthesist
- ⚫ **NULL** - The Skeptic
- ⚪ **GAIN** - Signal Amplifier
- ⬢ **SCHEMA** - Information Architect

## How to Use

1. Paste your content in the input box
2. Click "ACTIVATE SWARM"
3. Wait for all 8 personas to process
4. Export results in multiple formats

## Features

- Multi-perspective analysis
- Entity and relationship mapping
- Pattern recognition
- Critical skepticism
- Signal amplification
- **Information architecture recommendations** (new!)

## SCHEMA Persona

The newest addition! SCHEMA analyzes all outputs and recommends:
- Optimal file structures
- Best formats for each type of data
- How information should flow in a repository
- Relationship mapping between outputs

## Requirements

This Space requires an Anthropic API key (set in Space secrets as `ANTHROPIC_API_KEY`).
